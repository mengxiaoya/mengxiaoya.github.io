2. grep

3. awk

awk就是把文件逐行的读入，以空格为默认分隔符将每行切片，切开的部分再进行各种分析处理。awk语言的最基本功能是在文件或者字符串中基于指定规则浏览和抽取信息。通常，awk是以文件的一行为处理单位的。

4. sed

sed的作用就是实现对文件的 “ 增删改查 ” 。

sed遵循简单的工作流：

  1. 读取（从输入中读取某一行）
  2. 执行（在某一行上执行sed命令）
  3. 显示（把结果显示在输出中）

默认是显示修改后内容，不会修改原文件，除非使用-i 参数
