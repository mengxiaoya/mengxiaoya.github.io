程序员 - 活的是本事

发表于 2019-09-18 | 分类于 职业素养 |

一个程序员看待职业的方式，决定了他能走多远。有的人把职业当游戏，不论 buff 还是 bug 都要闯过去；有的人把职业当对手，不是一心想赢，只是不想输；有的人，活得是造型；有的人，活得是人设；程序员，活得是本事。本文阐述了程序员的20条职业感悟，希望大家互勉共进。

1. 关于 Work life balance

  1. 把身体锻炼好，只要你有一个健康的身体和健康的生活观，这个世界就饿不死人。

  2. 最好能够做到早起，一方面早上人的头脑比较清晰，另一方面可以在没有人打扰的时间里多学点东西。

  3. 自己的发展路线应该是最大限度地加强和发挥自己独特的组合，而不是寻求单项的超越。主要是通过实践，其次是要有意识地构造。

  4. 学习的能力和阅读的习惯，是从大学毕业后最该带到工作里的两样东西。

  5. 不要做个纯码农，多抽空做点其他自己感兴趣的事情。

2. 关于技术

  1. 编程是一件需要技术积累的工作。涉及的不仅仅是写程序，还有对系统的了解，对工具的熟悉，工作方法，经验很重要。

  2. 学习新技术的时候多看“官方文档”。编写文档的人群，通常就是这些技术或者软件的开发者，他们是对这些东西最了解的人。因此，官方文档质量是很高的，通常也是最新的。

  3. 把觉得不靠谱的需求放到最后做，也许到时候需求就变了。

  4. 宁愿终止程序也不要带着错误运行下去。

  5. 隔一段时间就回去看看自己几个月之前写的代码，这能够让你认识到代码的可读性有多重要，亲身体会才能懂得。

  6. 程序不等于数据结构加算法，而等于搜索引擎加英语。

3. 关于年龄

  1. 所谓的年龄歧视是在给自己设定心理障碍，你真正应该担心的不是年龄，在于你是否有职业规划，是否朝职业目标努力，你的年龄是否达到了相应的能力及经验。

  2. 如果你坚持持续学习，不管在哪一个行业都会越老越香的。

  3. 无论年龄多大，需要学习的东西有很多，在专业的道路上还是要继续尝试做技术创新。

  4. 条条大路通罗马，选择一个自己热爱的行业很重要。

4. 关于情商

  1. 心要皮实，但话语和脸皮要柔软，记住有句老话叫，伸手不打笑脸人。

  2. 编码的时候只思考同一个思维层次的逻辑，在这层完成之后再思考下一层。

  3. 千万别真的认为别人都是傻逼，否则会降低你的可合作性，可合作性在团队里是项很重要的能力。

  4. 真正的领导能力不是令人服从，而是让大家用正确高效的方法完成任务。

  5. 永远别小看程序媛。

最后，谈一点自己的想法。一定要记录自己的知识积累，才能获得技术的提升，不论是写文章还是做分享。或许你现在资历尚浅，技术思路和技术方向还很难摸清。这时学习与模仿对你来说至关重要，学习别人思考问题的角度，分析成熟的技术文章的知识脉络。Good Luck！
